# Hardware Design Overview

## Power Stage
- Three-phase inverter topology
- MOSFET and IGBT-based switching (iterative testing)
- RC low-pass filter for waveform smoothing

## Control Interface
- Arduino-based PWM generation
- Potentiometer-based frequency control
- Heat-sink assisted thermal protection

## Key Observations
- Smooth three-phase sinusoidal output achieved
- Transformer-less design improved efficiency
- Prototype voltage suitable for scaled testing
